Filter

El método .filter() crea un nuevo array con todos los elementos que cumplan la condición implementada por la función dada. Imagina que tienes una empresa con estos empleados (Nico, Karo y Behar) todos tienen una habilidad, pero a ti te interesa saber quien sabe JavaScript para dejarle una tarea, con .filter() puedes encontrarlo sin buscar uno por uno, filter, filtra el arrar y te devuelve uno nuevo con los objetos que cumplan con la condición

const empleados = [nico, karo, behar]
const KnowJavaScript = (empleados) => empleados.skill == 'JavaScript';
let skill = empleados.filter(KnowJavaScript)
console.log(skill)
